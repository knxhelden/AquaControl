# 🏊 Swimming Pool Control

An IoT-based pool controller for monitoring **pH** and **ORP** levels, measuring **water temperature**, and controlling external pool components such as **pumps** and **heating systems**.

## 🌊 Overview

This project is a custom-built pool automation system based on **ESP8266** controllers.  
Its main goal is to provide reliable **water quality monitoring** and basic **pool equipment control** in a modular and extensible way.

The system is designed around two separated measurement branches:

- 🧪 **pH measurement**
- ⚡ **ORP (Redox) measurement**

Each branch uses its own controller and ADC to improve signal stability and reduce interference during analog measurement.
